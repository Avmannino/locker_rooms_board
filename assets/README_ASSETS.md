# Required Asset Files

Please create the following image files in this directory:

## Sponsor Logo (4:1 aspect ratio)
- **presented_by.png** - Sponsor wordmark logo (recommended: 180x45px or 240x60px)

## Locker Brand Logos (1:1 aspect ratio)
- **locker_generic.png** - Generic placeholder for unsponsored lockers (recommended: 64x64px)
- **stateline_logo.png** - Stateline brand logo (recommended: 64x64px)
- **gsc_logo.png** - GSC brand logo (recommended: 64x64px)
- **gcds_boys_logo.png** - GCDS Boys brand logo (recommended: 64x64px)
- **gcds_girls_logo.png** - GCDS Girls brand logo (recommended: 64x64px)

## Notes
- All logos should be PNG format with transparent backgrounds
- Locker logos should be square (1:1 aspect ratio)
- Sponsor logo should be 4:1 aspect ratio (4 times wider than tall)
- Images will be automatically resized to fit the designated spaces
